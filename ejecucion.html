Ejecución del Programa

Guía completa para configurar y ejecutar el Sistema de Gestión de Cafetería

Requisitos del Sistema

Lo que necesitas para ejecutar correctamente el programa

Java Development Kit

Versión 8 o superior

Sistema Operativo

Windows, Linux o macOS

Terminal/Consola

Para ejecutar comandos

Archivos del Proyecto

Código fuente completo

Requisitos Detallados

Software Necesario:

  • Java Development Kit (JDK): Versión 8 o superior. Puedes descargarlo desde el sitio oficial de Oracle o usar OpenJDK.
  • Editor de texto o IDE: Opcional, pero recomendado para visualizar el código (VS Code, IntelliJ IDEA, Eclipse, etc.)
  • Terminal o línea de comandos: Para compilar y ejecutar el programa.

Archivos Necesarios:

  • Main.java - Clase principal del programa
  • GestionMesas.java - Módulo de gestión de mesas
  • MenuAlimentos.java - Módulo del menú de productos
  • GestionTareas.java - Sistema de gestión de tareas
  • GestionEmpleados.java - Módulo de empleados
  • EstadisticasVentas.java - Análisis de ventas
  • menu.csv - Archivo de datos del menú (debe estar en la misma carpeta)
Importante: Asegúrate de que todos los archivos .java estén en la misma carpeta y que el archivo menu.csv esté presente para el correcto funcionamiento del programa.

Guía de Ejecución desde tu Computadora

Cómo ejecutar el programa paso a paso

¿Qué archivo ejecutar?

El archivo principal que necesitas ejecutar para iniciar el programa es:

Archivo Principal: Main.java

Este es el punto de entrada del programa que contiene el método main() y gestiona el menú principal del sistema.

Estructura de archivos requerida:

carpeta-del-proyecto/
├── Main.java (ARCHIVO PRINCIPAL A EJECUTAR)
├── GestionMesas.java
├── MenuAlimentos.java
├── GestionTareas.java
├── GestionEmpleados.java
├── EstadisticasVentas.java
├── GestionArchivosEmpleados.java
├── DemoCafeteria.java
└── menu.csv (ARCHIVO DE DATOS DEL MENÚ)
1

Preparar el Entorno

Verifica que tienes Java instalado correctamente. Abre una terminal o línea de comandos y ejecuta:

java -version

Deberías ver información sobre la versión de Java instalada. Si no es así, instala Java antes de continuar.

2

Navegar al Directorio del Proyecto

Usa el comando cd para navegar a la carpeta donde guardaste los archivos del proyecto:

cd ruta/a/tu/carpeta/del/proyecto

Por ejemplo:

cd C:\Users\TuUsuario\Desktop\PowerCafe

Consejo: Puedes arrastrar la carpeta del proyecto a la terminal para obtener la ruta automáticamente.

3

Compilar el Programa

Compila todos los archivos Java del proyecto ejecutando:

javac *.java

Este comando compilará todos los archivos .java en la carpeta actual. Si hay errores de compilación, revísalos y corrige cualquier problema en el código.

Nota: Al compilar, se crearán archivos .class para cada archivo .java. Estos son los archivos ejecutables que Java necesita.
4

Ejecutar el Programa Principal

Una vez compilado correctamente, ejecuta el programa con:

java Main.java

Este comando ejecutará el método main() de la clase Main.java y el programa se iniciará mostrando el menú principal.

¡Listo! Si ves el menú principal del sistema, el programa se está ejecutando correctamente.
5

Usar el Programa

Una vez ejecutado, el programa mostrará un menú interactivo con las siguientes opciones:

  1. Gestión de Mesas - Para administrar mesas y pedidos
  2. Gestión de Productos - Para ver y gestionar el menú
  3. Gestión de Tareas - Para asignar y seguir tareas
  4. Gestión de Empleados - Para administrar el personal
  5. Análisis de Ventas - Para ver estadísticas y reportes
  6. Demostración - Ejecuta una demostración automática
  7. Salir - Cierra el programa

Selecciona la opción deseada ingresando el número correspondiente.

Solución de Problemas

Posibles errores y cómo solucionarlos

Errores Comunes

Error: "javac no se reconoce como un comando interno o externo"

Solución: Java no está instalado o no está configurado correctamente en el PATH del sistema. Instala Java JDK y configura las variables de entorno.

Error: "No se puede encontrar o cargar la clase principal Main"

Solución: El archivo Main.class no existe o no se compiló correctamente. Asegúrate de haber ejecutado javac *.java y de que Main.java esté en la carpeta actual.

Error: "FileNotFoundException: menu.csv"

Solución: El archivo menu.csv no está en la misma carpeta que los archivos .java. Asegúrate de que esté presente.

Error al procesar archivos CSV

Solución: Verifica que los archivos CSV tengan el formato correcto y estén codificados en UTF-8.

Consejo: Si experimentas problemas, ejecuta el programa en un IDE como IntelliJ IDEA o Eclipse, que proporciona mejores mensajes de error y depuración.

Comandos Alternativos

Compilar archivos individualmente

Si necesitas compilar archivos por separado:

javac Main.java javac GestionMesas.java javac MenuAlimentos.java javac GestionTareas.java javac GestionEmpleados.java javac EstadisticasVentas.java

Ejecutar con parámetros de memoria

Si el programa requiere más memoria:

java -Xmx512m Main.java

Ejecutar en modo verbose

Para ver información detallada de la ejecución:

java -verbose Main.java
