Guía por Módulos

Aprende a usar cada función del sistema

1

Gestión de Mesas

Este módulo te permite administrar las mesas de tu cafetería:

Funciones disponibles:
  • Crear Mesa: Agregar una nueva mesa al sistema
  • Eliminar Mesa: Remover una mesa existente
  • Agregar Comensales: Registrar clientes en una mesa
  • Realizar Pedido: Tomar órdenes de los clientes
  • Procesar Pedidos Pendientes: Preparar las órdenes en orden de llegada
  • Cerrar Mesa: Generar ticket y liberar la mesa
Ejemplo de uso: Cuando llegan clientes, creas una mesa, agregas los comensales, tomas su pedido, procesas la orden y finalmente cierras la mesa para generar el ticket de pago.
2

Gestión de Productos

Administra el menú de tu cafetería:

Funciones disponibles:
  • Ver Menú Completo: Mostrar todos los productos organizados por categorías
  • Buscar Producto: Encontrar productos por nombre o ID
  • Gestionar Pedido Individual: Crear un pedido simple sin mesa
  • Generar Ticket Individual: Crear factura para pedidos directos
Dato importante: El menú se carga automáticamente desde el archivo menu.csv. Los productos están organizados en categorías como Bebidas, Alimentos, Postres, etc.
3

Gestión de Tareas

Organiza el trabajo del personal:

Funciones disponibles:
  • Crear Tarea: Agregar nueva tarea con prioridad (1-5)
  • Procesar Tarea: Atender la tarea más urgente primero
  • Buscar Tarea: Encontrar tareas específicas
  • Distribuir Tareas: Repartir trabajo entre empleados
  • Ver Tiempo Total: Calcular tiempo de todas las tareas pendientes
Nota: Las tareas se procesan por prioridad (1 = más urgente, 5 = menos urgente). El sistema siempre atiende primero las tareas con mayor prioridad.
4

Gestión de Empleados

Administra la información del personal:

Funciones disponibles:
  • Agregar Empleado: Registrar nuevo empleado en el sistema
  • Eliminar Empleado: Remover empleado del sistema
  • Buscar Empleado: Encontrar empleado por ID
  • Mostrar Empleados: Ver lista completa ordenada por ID
  • Calcular Nómina: Ver total de salarios del personal
  • Filtrar por Departamento: Ver empleados por área
Ventaja: El sistema usa un árbol binario que hace las búsquedas muy rápidas, incluso con muchos empleados registrados.
5

Análisis de Ventas

Obtén reportes y estadísticas de tu negocio:

Funciones disponibles:
  • Analizar Ventas del Día: Revisar todos los tickets del día
  • Ordenar por Monto: Ver ventas de mayor a menor
  • Buscar Venta Específica: Encontrar ticket por monto
  • Productos Más Vendidos: Ver tendencias de productos populares
  • Total del Día: Calcular ingresos totales
Automatización: Los tickets se guardan automáticamente cuando cierras una mesa. El análisis de ventas lee estos archivos para generar los reportes.

Ejemplo de Uso en un Día Típico

1

Apertura

Al abrir la cafetería, revisa las tareas pendientes y asígnalas al personal correspondiente usando el módulo de Gestión de Tareas.

2

Servicio

Durante el servicio, usa el módulo de Gestión de Mesas para atender a los clientes: crear mesas, tomar pedidos y procesar órdenes.

3

Cierre

Al final del día, usa el módulo de Análisis de Ventas para revisar el desempeño del día y generar reportes.

Solución de Problemas Comunes

¿Tienes problemas usando el sistema?

Problemas Frecuentes

No puedo encontrar un producto en el menú

Solución: Usa la función "Buscar Producto" en el módulo de Gestión de Productos. Puedes buscar por nombre o ID.

Las tareas no se están procesando en el orden correcto

Solución: Recuerda que las tareas se procesan por prioridad (1-5). Verifica que estés asignando las prioridades correctamente.

No encuentro un empleado en el sistema

Solución: Usa la función "Buscar Empleado" ingresando el ID exacto. También puedes ver la lista completa ordenada.

Los tickets no se están generando

Solución: Asegúrate de usar la opción "Cerrar Mesa" en el módulo de Gestión de Mesas para generar el ticket automáticamente.

Recuerda: Si experimentas problemas técnicos con el programa mismo (no con su uso), consulta la sección "Ejecución" para solución de problemas de instalación y configuración.
